                                                Zeldiablo est le <strong>premier projet Java avec interface graphique</strong> que j'ai réalisé. Il a été fait dans le cadre de mes études en DUT Informatique à l'IUT Charlemagne
                                            </p>
                                            <p style="text-align: left;">
                                                L'objectif était de créer un <strong>jeu Java avec le moteur graphique qui nous était donné</strong>. Ici nous avons opté pour un labyrinthe avec des monstres
                                            </p>
                                            <p style="text-align: left;">
                                                Le labyrinthe est constitué de plusieurs étages (prédéfinis), le joueur contrôle <strong>Megaman</strong> et doit éviter les monstres.                                             </p>
                                            <p style="text-align: left;">
                                                Pour réussir le jeu il faut aller jusqu'à la <strong>case trophée</strong> se trouvant au dernier étage, cependant nous avons également ajouter une case invisible qui fait gagner le joueur, celle-ci fait apparaître <strong>Shrek !</strong>
                                            </p>
